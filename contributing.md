# Contribution Guidelines

Bitte stelle sicher, dass dein Pull Request folgende Punkte erfüllt:

- Das Tool ist aktiv in Entwicklung und wird gewartet
- Ein Link zur Webseite oder Dokumentation ist vorhanden
- Eine kurze, prägnante Beschreibung auf Deutsch ist enthalten
- Das Tool ist relevant für den DACH-Markt (DE/AT/CH)
- Kostenloser Zugang oder Testzeitraum ist vorhanden
- Duplikate werden nicht eingereicht

Danke für deinen Beitrag!
